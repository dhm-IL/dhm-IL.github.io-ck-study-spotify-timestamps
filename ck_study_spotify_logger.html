How to use

  1. Create a Spotify developer app and add this page URL as a Redirect URI.
  2. Optionally hardcode fixedClientId and fixedRedirectUri near the top of this file before deployment so collaborators never have to type them.
  3. Click Connect Spotify and log in with the participant's own Spotify Premium account.
  4. Open Spotify somewhere on that same account so there is an active device, then use Load Devices if you want to target one explicitly.
  5. Use Load My Playlists to pick one of the logged-in user's playlists, or paste a playlist URL manually.
  6. Enable the microphone if you want room-noise estimates.
  7. Enter session metadata, then click Start Session + Play Playlist.
  8. When done, click End Session, fill end fields if needed, then download JSON or CSV and email the file.
